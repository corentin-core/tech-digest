# tech-digest

Unlisted, `noindex` HTML pages for **personal tech-news digests**, delivered to a Kobo
eReader via Instapaper.

Generated by the `tech-digest` Claude Code skill: it fetches tech sources (TLDR Dev, This
Week in Rust, Console.dev, Hacker News, Lobste.rs, systems blogs), filters them for a
backend / systems / performance profile, distills the keepers, and publishes each digest
here. The page URL is then added to Instapaper via its Simple API, and the Kobo syncs it
under *My Articles*.

## URLs

Pages are served by GitHub Pages (source: `main` branch, `/docs` folder) at:

    https://corentin-core.github.io/tech-digest/<slug>.html

| Type    | Slug                      | Example                     |
|---------|---------------------------|-----------------------------|
| Daily   | `YYYY-MM-DD.html`         | `2026-06-08.html`           |
| Weekly  | `YYYY-MM-DD-weekly.html`  | `2026-06-08-weekly.html`    |

Each page carries `<meta name="robots" content="noindex, nofollow">`. The content is
distilled **public** tech news (nothing private), but pages are unlisted and not indexed.

The markdown source of every digest is archived locally in `~/Twake/Perso/tech-digest/`.
